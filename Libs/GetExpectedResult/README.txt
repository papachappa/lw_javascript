How to get expected result:

1. TV set should be in required state:
If you want to get conflicted services, LCN conflict should be displayed.
If you need new found services, result of Auto scan should be displayed and so on.
2. Prepare print script:
Open required script and insert correct parameters, list name and media type for example.
3. Prepare WeBiz to start: 
Open runWeBiz.sh file and update TV IP and script name, save update.
4. Run runWeBiz.sh in terminal - result will be print into terminal window.
5. Close terminal.
